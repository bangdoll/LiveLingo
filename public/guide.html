🔑 LiveLingo API Key 設定教學

三步驟完成設定，開始使用即時雙語語音助理

為什麼需要 API Key？

LiveLingo 使用 OpenAI 的 Realtime API 提供即時語音對話、雙語字幕翻譯與網路搜尋功能。 您可以選擇使用系統預設金鑰（額度有限），或填入自己的 OpenAI API Key 以獲得完整的使用體驗。

💡 自備金鑰的好處 使用自己的 API Key 可以享有更高的使用額度、更低的延遲，且不受系統共用配額的限制。 金鑰僅儲存在您的瀏覽器本地 (localStorage)，不會傳送至任何第三方伺服器。

設定步驟

步驟一：建立 OpenAI 帳號

  1. 前往 OpenAI Platform 註冊帳號
  2. 完成電子郵件驗證與手機驗證
  3. 登入後進入 Billing 頁面，加入付款方式（信用卡或金融卡）

步驟二：產生 API Key

  1. 前往 API Keys 頁面
  2. 點擊 「Create new secret key」
  3. 為金鑰取個名稱（例如 LiveLingo），選擇權限為 All
  4. 複製產生的金鑰（格式為 sk-proj-...）
⚠️ 安全提醒 API Key 只會在建立時顯示一次，請立即複製並妥善保管。切勿將金鑰分享給他人或張貼在公開場合。

步驟三：在 LiveLingo 中填入金鑰

  1. 回到 LiveLingo 首頁
  2. 點擊右上角的 ⚙ 齒輪圖示 打開設定面板
  3. 在「OpenAI API Key」欄位中貼上您的金鑰
  4. 按下 Enter 或點擊其他地方，金鑰會自動儲存
  5. 點擊「開始對話」即可使用！

常見問題

Q：金鑰儲存在哪裡？安全嗎？

金鑰儲存在您瀏覽器的 localStorage 中，只存在您的裝置上。建立 Token 時會透過 LiveLingo 伺服器中轉（因為瀏覽器無法直接呼叫 OpenAI 的 API 建立 Token），但金鑰本身不會被記錄或儲存於任何伺服器。

Q：不填 API Key 也能用嗎？

是的，如果伺服器已設定系統金鑰，您可以直接使用。但系統金鑰的額度有限，建議長期使用者填入自己的金鑰。

Q：如何移除我的金鑰？

打開 ⚙ 設定面板，清空 API Key 欄位後按 Enter 即可。也可以在瀏覽器開發者工具中執行 localStorage.removeItem("livelingo_api_key")。

Q：什麼是「即時口譯模式」？

這是 LiveLingo 的進階功能，使用 OpenAI 最新的 gpt-realtime-translate 模型。 啟用後，語音翻譯不再需要「語音→文字→翻譯」三步串接，而是直接在語音層面即時口譯， 延遲大幅降低，體驗就像現場口譯員一樣流暢。
